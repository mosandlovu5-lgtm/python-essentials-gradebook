# Description
A console program that stores students and their marks in memory and shows statistics such as averages, the top student, and pass/fail lists.

# How to run
python gradebook.py

# Features
1. Add a student
2. Add a mark
3. View all students
4. Student summary
5. Class statistics
6. Remove a student
7. Exit
